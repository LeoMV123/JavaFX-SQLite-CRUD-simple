# Ejercicio JavaFX: Agenda electrónica

Se necesita un programa para gestionar los contactos de una **agenda electrónica**

1. En la agenda se debe permitir ingresar los siguientes datos: **Nombre**, **Apellido Paterno**, **Apellido Materno** y **Teléfono**.
2. Se necesita almacenar datos de 10 personas.
3. El programa debe permitir guardar los valores.
